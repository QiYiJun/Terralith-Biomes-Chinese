# Terralith Biomes Languages
  一个Terralith的生物群系名称语言材质包

  配合地图模组使用可获得更好的生物群系名称显示

# Update content & Tips
  - 绝大部分语言翻译内容为官方
  - 本人做了细微的调整、添加部分缺失的词条和翻译(仅简体中文，其他语言我不会呀!)
  - 如果出现`\u0041`类似的字符，请下载和使用Unicode的版本
